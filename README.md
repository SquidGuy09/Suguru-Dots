# Suguru Dots 🦑
Smooth Hyprland rice with Suguru Geto wallpaper.

## Components
- **WM:** Hyprland
- **Bar:** Waybar
- **Terminal:** Kitty
- **Launcher:** Rofi
- **Notifications:** Swaync
- **Fetch:** Fastfetch
